外出時に自宅から一定距離（例：20m）離れた瞬間、プッシュ通知で忘れ物の確認を促すWebアプリです。

## 🚀 主な機能
* **データ永続化 (Cookie管理)**: 緯度・経度・通知距離・チェックリストの中身をすべてCookieに保存します。サイトをリロードしたりブラウザを閉じたりしても、入力したデータは保持されます。
* **常時表示チェックリスト**: 外出先でも家の中でも、常に登録したメモ（忘れ物リスト）を確認・編集できます。
* **位置連動プッシュ通知**: 指定した距離以上離れると、サービスワーカーを通じて「忘れ物はないですか？」という通知をデバイスに送ります。
* **レスポンシブ・ダークデザイン**: 外出時の視認性を考慮した、暗所に強いモダンなユーザーインターフェース。

🛠 使い方
1. 初期設定

    場所の設定: 「場所の設定を変更」を開き、「現在地を取得」をクリックして自宅の座標を入力します。

    距離の設定: 通知を飛ばしたい距離（例：20）を入力し、「設定を保存」をクリックします。

2. リスト登録

    アイテム追加: 「忘れ物リスト」セクションの入力欄に、持ち物を入力して「追加」します。

    表示: 追加されたアイテムはリストとして画面に常時表示されます。

3. 通知の準備

    権限の許可: ブラウザから通知の許可を求められたら「許可」を選択してください。

4. 動作確認

    通知の実行: アプリを開いた状態で自宅から設定距離以上離れると、プッシュ通知が届きます。

    完了処理: 持ち物を確認し、完了したものはチェックボックスを入れるとリストから自動的に削除されます。

📋 技術スタック

    Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

    Storage: document.cookie (有効期限：365日)

    APIs: Geolocation API, Web Notification API, Service Worker API

    Algorithm: Haversine Formula（球面距離計算）

⚠️ 注意事項

    HTTPS必須: 位置情報および通知機能の利用には、セキュリティ保護された通信（HTTPS）環境が必要です。

    バックグラウンド動作: ブラウザの仕様により、タブを完全に閉じていると通知が届かない場合があります。ブラウザをバックグラウンド（タブを開いた状態）にしておくことを推奨します。

    精度: GPSの精度により、距離の計算に数メートルの誤差が生じる場合があります。
